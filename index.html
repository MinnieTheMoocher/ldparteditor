LDPartEditor

Create and edit LDraw Parts quick and easy.

Download this project as a .zip file Download this project as a tar.gz file

Description

The LDPartEditor is the platform independent CAD tool for LDraw parts.

Minimum System Requirements:

  • Java Runtime Environment (JRE) 1.6
  • OpenGL 2.1 compatible Graphic Card
  • Operating System (32/64bit): Windows [XP; Vista; 7 or newer], Mac OS X [>=10.6], Linux [e.g. Ubuntu Linux >=10.4]
  • CPU: Intel Pentium 4, AMD Athlon 64 or newer..
  • RAM: 1000 MB?
  • Video-Memory: 256-512 MB?
  • Free Disk Space: 25 MB?

Recommended Requirements:

  • Operating System (32/64bit): Windows 7, Mac OS X [>=10.6], Linux [e.g. Ubuntu Linux >=10.4]
  • CPU: Multicore-Processor e.g. Intel Core 2 Duo or AMD Athlon II (>2.0Ghz)
  • RAM: >1.500 MB
  • Video-Memory: >512 MB
  • Free Disk Space: 250 MB
